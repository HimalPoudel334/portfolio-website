About Me

Hello! I'm Himal Poudel, a dedicated backend developer and database administrator with a passion for building robust, scalable server-side applications and optimizing database performance. My journey into backend development began with a fascination for how data flows through systems, and it has since grown into expertise in creating efficient APIs and database architectures.

I'm passionate about DevOps practices and believe in continuous integration and deployment for reliable software delivery.
